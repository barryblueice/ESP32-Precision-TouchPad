This folder includes all the PCBs of TouchPad.

 - `Gerber`: Gerber files are PCB manufacturing files. <br>Impedance control standard: `JLC04161H-3313`, choose 1.0mm for thickness if ordering PCB.
 - `Project`: Project is the PCB projects designed by EasyEDA Pro Edition.
